# ros_caffe
================
## Purpose
Integrate Caffe model into ROS

## Requirement
1 Enviornment: Ubuntu 14.04 and catkin build
2 Download the model cafe by yourself

## Install
$ cd $CATKIN_WS/src
$ git clone https://github.com/tzutalin/ros_caffe.git

## Build
$ cd $CATKIN_WS
$ catkin_make

## Run
$ rosrun ros_caffe ros_caffe_test
